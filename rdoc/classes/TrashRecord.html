empty_trash(max_age, options = {})

Empty the trash by deleting records older than the specified maximum age. You can optionally specify :only or :except in the options hash with a class or array of classes as the value to limit the trashed classes which should be cleared. This is useful if you want to keep different classes for different lengths of time.
